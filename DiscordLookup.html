Get more out of Discord with Discord Lookup

Snowflake Decoder Get the creation date of a Snowflake, and detailed information about Discord users and guilds.
User Lookup Get detailed information about Discord users with creation date, profile picture, banner and badges.
Guild Lookup Get detailed information about Discord guilds with creation date, Invite/Vanity URL, features and emojis.
Application Lookup Get detailed information about Discord applications with description, links, tags and flags.
Guild List Show all the guilds you are on, with counters, permissions, features, experiments and more information about the guilds.
Discord Experiments All Discord Client & Guild Experiments with Rollout Status and detailed information about Treatments, Groups and Overrides.
Invite Resolver Get detailed information about every invite and vanity url including event information.
Timestamp Styles Generate Discord timestamp styles based on a date, time, snowflake or timestamp.
Permissions Calculator Calculate Discord permissions integer based on the required bot permissions.
Snowflake Distance Calculator Calculate the distance between two Discord Snowflakes.
Guild Shard Calculator Calculate the Shard ID of a guild using the Guild ID and the total number of shards.
Discord Webhook Invalidator Immediately delete a Discord webhook to eliminate evil webhooks.
Bad Domain Check Check if a domain is listed as a bad domain on Discord.
MurmurHash3 Calculator Calculate the MurmurHash3 of a string with an optional Mod 10000.
